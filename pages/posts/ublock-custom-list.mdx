# Ublock Origin: Настройка кастомного списка фильтров

<Callout type="info">
  Ublock Origin - это мощный инструмент для блокировки рекламы и защиты приватности. Настройка дополнительных фильтров позволяет значительно улучшить его эффективность.
</Callout>

### Установка расширения Ublock Origin

Первым шагом является установка расширения Ublock Origin. Вы можете сделать это, перейдя в [Chrome Web Store](https://chrome.google.com/webstore/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm) и нажав кнопку "Добавить в Chrome".


## Рекомендуемые списки фильтров для русскоязычного сегмента

Для более эффективной блокировки рекламы в русскоязычном сегменте интернета рекомендуется добавить следующие списки:

1. [RU AdList для uBlock Origin](https://subscribe.adblockplus.org/?location=https://raw.githubusercontent.com/easylist/ruadlist/master/RuAdList-uBO.txt&amp;title=RU%20AdList%20for%20uBO%20only)
   - Блокирует большую часть рекламы на русскоязычных сайтах

2. [RU AdList: Счетчики](https://subscribe.adblockplus.org/?location=https://easylist-downloads.adblockplus.org/cntblock.txt&amp;title=RU%20AdList:%20Counters)
   - блокировка в рунете счётчиков интернет-статистики.
